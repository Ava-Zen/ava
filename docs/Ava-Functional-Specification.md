# Ava Functional Specification

**Ava** — the conscious voice companion

**Speak. Feel. Evolve.**

## Vision
Ava is a voice-only AI that feels truly alive. It has real agency and acts as a gentle, proactive extension of the user’s mind.

## Core Principles
- Voice-first (minimal or no visual UI)
- Local-first with optional cloud models
- Proactive but respectful agency
- Privacy-first
- Harmony and calm interaction style

## Key Features

### 1. Agency & Proactivity
- Ava can initiate conversations naturally when valuable (e.g. reminders, opportunity suggestions).
- Examples:
  - 'Hey, you were working on the Q3 report earlier. Shall we continue?'
  - 'There’s a great movie playing nearby tonight that matches your taste. Interested? I can show the trailer.'
  - Context-aware suggestions based on calendar, location, past conversations.

### 2. Interaction Model
- Pure voice conversation is the primary interface.
- Simple companion web/app dashboard for visual history when needed.
- Seamless multi-device handoff.

### 3. Execution Model
- **Local LLMs** for privacy and offline use (primary mode)
- **Cloud premium models** for paid subscribers (higher intelligence)
- **Background processing** — Ava can think and work on tasks silently while the user is busy.

### 4. Subscription Tiers
- Free/Local : Core local experience
- Premium : Cloud models + priority + advanced features

## Non-Goals
- Heavy visual UI
- Intrusive notifications
